# 🔥 Crypto Roast

Get roasted based on your on-chain activity. Built for Farcaster Mini Apps.

## Features

- 🔗 Connect wallet (Base mainnet)
- 📊 Analyze on-chain activity
- 🔥 Generate savage roast
- 💾 Download shareable card
- 🚀 Share directly to Farcaster

## How it works

1. Connect your wallet
2. We analyze: balance, tx history, NFTs
3. AI generates personalized roast
4. Share the pain with your friends

## Roast Categories

- 💀 **Certified Degen** - High-risk, high-loss trader
- 🐋 **Sad Whale** - Rich but anxious
- 📝 **Paper Hands** - Barely active
- 🖼️ **JPEG Degen** - NFT obsessed
- 🎯 **Mid Degen** - Average degen behavior

## Tech Stack

- Next.js 14
- Tailwind CSS
- RainbowKit
- wagmi/viem
- Base Network

## Deploy

1. Deploy contract (optional, for leaderboard)
2. Update `farcaster.json` with your domain
3. Deploy to Vercel
4. Submit to Farcaster Mini Apps
